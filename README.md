# AI Trading Agents Dashboard

A real-time dashboard monitoring 4 AI-powered trading agents:

- **Warren** - Value investing (inspired by Warren Buffett)
- **George** - Macro trading (inspired by George Soros)
- **Ray** - Systematic/Risk parity (inspired by Ray Dalio)
- **Cathie** - Crypto ETFs (inspired by Cathie Wood)

## Features

- 📊 Real-time portfolio tracking
- 📈 Interactive Plotly charts
- 💰 Holdings and transactions display
- 📝 Live activity logs

## Environment Variables Required

Set these in your Space's Settings → Repository Secrets:

- `GOOGLE_API_KEY` - Google AI API key
- `POLYGON_API_KEY` - Polygon.io API key
- `BRAVE_API_KEY` - Search API key
